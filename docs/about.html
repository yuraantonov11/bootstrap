Історія

Спочатку створений дизайнерами та розробниками Twitter, Bootstrap став одним з найпопулярніших front-end фреймворком та проектом з відкритим вихідним кодом у світі.

Bootstrap створений @mdo та @fat в Twitter всередині 2010 року. Перед тим як стати фреймворком з відкритим вихідним кодом, Bootstrap був відомий як Twitter Blueprint. Після кількох місяців розробки Twitter провела first Hack Week і проект вибухнув, оскільки розробники всіх рівнів поринули в нього без будь-якого зовнішного керівництва. Він слугував в якості посібника по стилю для внутрішньої розробки компанії більше року до його публічного релізу, та продовжує це робити й сьогодні.

Починаючи з випуску у П'ятницю, 19 серпня, 2011, ми мали більше двадцяти релізів, у тому числі два основних переписування до v2 та v3. У Bootstrap 2, ми додали функціонал чутливості лише в якості додаткових стилів. Грунтуючись на цьому в Bootstrap 3, ми переписали бібліотеки ще раз, щоб зробити його чутливим за промовчанням, означивши наш підхід – мобільність повинна бути на першому місті.

Назва

Проект та фреймворк повинні завжди називатись Bootstrap. Без попереднього слова Twitter, без великої букви s, та без будь-яких абревіатур, за виключенням однієї - великої літери B.
