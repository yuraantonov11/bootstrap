Спадаюче меню

Контекстні меню, що перемикаються, для відображення списку лінків. Їхня інтерактивність забезпечується JavaScript плагіном для спадаючого меню.

Приклад

Вкладіть кнопку-перемикач та його меню всередину елемента з класом .dropdown, або в інший елемент, котрий оголошує position: relative;.

Опції вирівнювання

За промовчанням, спадаюче меню автоматично позиціоновано 100% згори та в лівий кут батьківського елемента. Додайте .dropdown-menu-right до .dropdown-menu щоб вирівняти праворуч спадаюче меню.

Може потребувати додаткового позиціонування

Спадаюче меню автоматично позиціонується через CSS всередині звичайного потоку документа. Це означає, що спадаюче меню може обрізатись батьківським елементом, якщо встановлено певну властивісь overflow, або з'являтись за межами оглядового вікна. Ці проблеми ви можете вирішити самостійно по мірі їх виникнення.

Застаріле вирівнювання .pull-right

As of v3.1.0, we've deprecated .pull-right on dropdown menus. To right-align a menu, use .dropdown-menu-right. Right-aligned nav components in the navbar use a mixin version of this class to automatically align the menu. To override it, use .dropdown-menu-left.
