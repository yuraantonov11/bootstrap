Bootlint

Bootlint це офіційний інструмент Bootstrap HTML linter. Він автоматично робить перевірку на наявність ряду відомих HTML-помилок на сторінках, що використовуються Bootstrap в достатньо "правильний" спосіб. Компоненти чи віджети Vanilla Bootstrap вимагають щоб їхні частини DOM відповідали певній структурі. Bootlint перевіряє, що компоненти Bootstrap мають коректну структуру HTML. Розгляньте можливість додати Bootlint до інструментів веб-розробки Bootstrap, щоб жодна із відомих помилок не сповільнювала розробку вашого проекта.
