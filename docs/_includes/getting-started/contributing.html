Співпраця

Маєте бажання допомогти з українським перекладом Bootstrap? Прекрасно =).

Зараз, українським перекладом Bootstrap та підтримкою сайта займається одна людина причому зовсім не верстальник сайтів та зовсім не перекладач. На момент написання даного тексту, вийшла нова версія 3.3.1, а тому на сайті дещо додалось неперекладених фрагментів. Без вашої допомоги, ця одна людина легко "зашиїться", бо займається цим у вільний від роботи час, а тому матимемо застарілий чи неповний переклад. Тож співпраця в цьому плані дуже вітається.

Як можна допомогти? Покищо це можна зробити лише через відкриття Pull Requests на Github
