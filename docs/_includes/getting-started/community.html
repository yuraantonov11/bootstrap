Спільнота

Дізнавайтесь про розробку Bootstrap та звертайтесь до його спільноти за допомогою цих корисних ресурсів.

  • Читайте та підписуйтесь на The Official Bootstrap Blog.
  • Чат з іншими прихильниками Bootstrap через IRC на сервері irc.freenode.net, в ##twitter-bootstrap channel.
  • Запитувати про допомогу стосовно Bootstrap можна на StackOverflow використовуючи тег twitter-bootstrap-3.
  • Знайдіть захоплюючі приклади проглядаючи виставку Bootstrap Expo, де показані роботи з використанням Bootstrap.

Ви також можете підписатись на @twbootstrap on Twitter щоб знати про останні плітки та захоплююче музикальне відео.
