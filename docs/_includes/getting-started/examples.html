Приклади

На базі шаблона, що надано вище, Bootstrap створює багато компонентів. Ми заохочуємо вас, щоб ви підбирали та адаптували Bootstrap під ваші власні проекти.

Використання фреймворка

Стартовий шаблон

Нічого окрім базового: збірка CSS, JavaScript, та використано контейнер.

Теми Bootstrap

Налаштуйте теми Bootstrap щоб урізноманітнити вигляд сторінок.

Сітки

Декілька прикладів макетів сітки з усіма чотирма рівнями, вкладеності та інше.

Великий екран

Шаблон з великим екраном та навпанеллю і з кількома колонками сітки.

Вузький екран

Створення більш персоналізованих сторінок через звуження контейнера та великого екрану.

Статичне навігаційне меню

Дуже базовий шаблон, який включає навігаційне меню разом з додатковим вмістом.

Фіксоване навігаційне меню

Дуже базовий шаблон, який включає навігаційне меню разом з додатковим вмістом.

Підбірка компонентів

Cover

Односторінковий шаблон для побудови простої та гарної домашньої сторінки.

Карусель

Поєднання в шаблоні нав. меню та каруселі, з додаванням деяких нових компонентів.

Блог

Простий шаблон блогу на дві колонки з підібраною навігацією, заголовком, та типом.

Панель керування

Базова структура для адмінської панелі керування з фіксованою навпанеллю.

Сторінка входу

Шаблон користувальницької форми налаштований для авторизації на сайті.

Вирівняне навменю

Створення компонованого нав. меню з вирівняними лінками. Увага! Не дуже дружить з Safari.

Закріплений колонтитул

Закріплення колонтитулу в нижній частині браузера для випадку, коли висота контента сама собою не робить цього.

Закріплений колонтитул з навменю

Прикріплено колонтитул у нижню частину браузера, з навменю у верхній частині.
